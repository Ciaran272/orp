## 📦 安装方法

### 方法一：自动安装（推荐 - Windows用户）

1. **下载扩展文件夹**
   - 解压到任意位置（建议放在不会删除的文件夹）

2. **运行安装程序**
   - 双击 `一键安装.bat` 文件
   - 按照提示操作即可

### 方法二：手动安装（所有用户）

#### Chrome 浏览器

1. **打开扩展管理页面**
   - 在地址栏输入：`chrome://extensions/`
   - 或点击浏览器右上角 ⋮ → 更多工具 → 扩展程序

2. **启用开发者模式**
   - 打开页面右上角的"开发者模式"开关

3. **加载扩展**
   - 点击左上角"加载已解压的扩展程序"
   - 选择解压后的 `jietu` 文件夹
   - 点击"选择文件夹"

4. **完成安装**
   - 扩展图标会出现在浏览器工具栏
   - 可以点击图标开始使用

#### Edge 浏览器

1. **打开扩展管理页面**
   - 在地址栏输入：`edge://extensions/`
   - 或点击浏览器右上角 ⋯ → 扩展

2. **启用开发者模式**
   - 打开页面左下角的"开发人员模式"开关

3. **加载扩展**
   - 点击"加载解压缩的扩展"
   - 选择解压后的 `jietu` 文件夹
   - 点击"选择文件夹"

4. **完成安装**
   - 扩展图标会出现在浏览器工具栏

## 🚀 快速开始

### 1. 打开扩展

**方式一：点击图标**
- 点击浏览器工具栏上的扩展图标 📸

**方式二：使用快捷键**
- Windows/Linux: `Ctrl + Shift + S`
- Mac: `Command + Shift + S`

### 2. 选择截图模式

扩展提供4种截图模式：

#### 🤖 自动识别模式（默认）
- 自动识别网页中的卡片元素
- 适合有规律的卡片布局页面
- 可以批量截取多个卡片

#### 👆 手动选择模式
- 点击网页上的任意元素进行截图
- 适合自由选择特定内容
- 支持预览和手动下载

#### 🔍 CSS选择器模式
- 输入CSS选择器精确定位元素
- 适合开发者和高级用户
- 默认选择器：`.main-card`

#### ✂️ 自由截图模式
- 鼠标拖拽框选任意区域
- 类似QQ截图工具
- 最灵活的截图方式

### 3. 设置截图选项

- ✅ **高清模式** - 2倍分辨率，更清晰（推荐开启）
- 🎨 **透明背景** - 保留透明背景（PNG格式）
- 💾 **自动下载** - 自动保存到下载文件夹
- 📁 **文件命名** - 在设置页面自定义（支持模板变量）

### 4. 开始截图

- 点击 **"📸 开始截图"** 按钮
- 或点击其他辅助按钮：
  - **🖼️ 截取可见区域** - 截取当前屏幕内容
  - **📄 截取整个页面** - 截取完整网页（包括滚动部分）

## ⚙️ 高级设置

### 进入设置页面

1. 点击扩展图标打开弹窗
2. 点击底部的 **"⚙️ 设置"** 链接

### 文件命名设置

#### 文件名模板
支持以下变量：
- `{n}` - 序号（如：1, 2, 3...）
- `{date}` - 日期（如：20241016）
- `{time}` - 时间（如：143520）

**示例**：
- `截图_{n}` → 截图_1.png, 截图_2.png
- `第{n}张` → 第1张.png, 第2张.png
- `截图{n}_{date}` → 截图1_20241016.png
- `{date}_{time}_{n}` → 20241016_143520_1.png

#### 起始序号
- 设置截图编号从几开始
- 默认：1
- 范围：0 - 9999

#### 数字补零
- **不补零**：1, 2, 3, ...
- **2位补零**：01, 02, 03, ...
- **3位补零**：001, 002, 003, ...

### 图片质量设置

- **标准（1倍）** - 原始尺寸，文件小
- **高清（2倍）** - 2倍分辨率，推荐
- **超高清（3倍）** - 3倍分辨率，文件大

> 💡 **提示**：勾选弹窗中的"高清模式"才会应用此设置

### 图片格式设置

- **PNG** - 无损格式，支持透明背景，文件较大
- **JPG** - 有损格式，文件较小，不支持透明

## 💡 使用技巧

### 技巧1：批量截图卡片
1. 打开包含多个卡片的网页（如商品列表、新闻列表）
2. 选择"自动识别"模式
3. 点击"开始截图"
4. 扩展会自动识别并截取所有卡片

### 技巧2：精确截图
1. 选择"CSS选择器"模式
2. 按F12打开开发者工具
3. 找到目标元素的CSS类名（如 `.product-card`）
4. 在扩展中输入该选择器
5. 点击"开始截图"

### 技巧3：截取长图
1. 打开要截图的网页
2. 点击扩展图标
3. 直接点击"📄 截取整个页面"
4. 等待处理完成（长页面需要较长时间）

### 技巧4：自由框选
1. 选择"自由截图"模式
2. 点击"开始截图"
3. 在网页上拖拽鼠标框选区域
4. 松开鼠标自动截图
5. 按ESC键可以取消

### 技巧5：设置快捷键
1. 打开设置页面
2. 点击"🚀 打开快捷键设置"
3. 为扩展设置自定义快捷键
4. 推荐：`Ctrl+Shift+S` 或 `Alt+S`

## ❓ 常见问题

### Q1: 为什么点击扩展图标没反应？

**A**: 可能是以下原因：
- 当前页面是系统页面（chrome://、edge:// 等）
- 页面刚加载，扩展脚本未初始化
- **解决方法**：刷新网页（F5）后重试

### Q2: 自动识别找不到元素怎么办？

**A**: 三种解决方案：
1. 切换到"手动选择"模式，点击想要截图的元素
2. 切换到"CSS选择器"模式，输入准确的选择器
3. 切换到"自由截图"模式，手动框选区域

### Q3: 截图质量不清晰？

**A**: 检查以下设置：
1. 确保勾选了"高清模式"复选框
2. 在设置页面将"图片质量"设为"高清（2倍）"或"超高清（3倍）"
3. 选择PNG格式（比JPG清晰）

### Q4: 截图包含了不想要的内容？

**A**: 使用更精确的截图方式：
1. 使用"手动选择"模式，只截取需要的元素
2. 使用"自由截图"模式，精确框选需要的区域
3. 使用"CSS选择器"模式，输入更精确的选择器

### Q5: 下载的文件在哪里？

**A**: 
- 默认保存在浏览器的下载文件夹
- Chrome: `C:\Users\你的用户名\Downloads\`
- 可以在浏览器设置中更改下载位置

### Q6: 可以截取跨域图片吗？

**A**: 
- 大部分网站可以正常截图
- 部分启用严格CORS的网站可能无法截取
- 这是浏览器安全限制，无法绕过

### Q7: 批量截图序号会重复吗？

**A**: 
- 不会！扩展使用连续编号
- 同一会话中多次截图，序号会自动递增
- 刷新页面后序号会重置

### Q8: 如何卸载扩展？

**A**: 
1. 打开 `chrome://extensions/` 或 `edge://extensions/`
2. 找到"智能卡片截图助手"
3. 点击"移除"按钮
4. 确认卸载

## 🔧 故障排除

### 问题：扩展安装后无法使用

**排查步骤**：
1. 确认所有必需文件都存在
2. 刷新扩展（在扩展管理页面点击刷新图标）
3. 刷新网页（F5）
4. 检查控制台是否有错误信息（F12）

### 问题：截图失败

**排查步骤**：
1. 确认不是在系统页面（chrome://）
2. 检查网络连接（某些图片需要加载）
3. 尝试降低图片质量设置
4. 使用更简单的截图模式（如"自由截图"）

### 问题：文件命名不符合预期

**排查步骤**：
1. 打开设置页面检查模板设置
2. 确认变量格式正确（`{n}` 而不是 `{N}`）
3. 点击"恢复默认"后重新设置
4. 保存设置后重新截图
